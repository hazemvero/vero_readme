Admin Dashboard

Powerful Admin Dashboard
Built with React & Supabase

The admin dashboard is a modern web panel built using React and Supabase for real-time, secure management.
It allows you to:

  • Add and send new notifications to users
  • Add new tokens to the wallet
  • Create new admin accounts
  • Review and approve withdrawal requests from in-app games
  • Manage all app data securely and efficiently

All actions are performed in real-time with a user-friendly interface, giving you full control over your platform.
